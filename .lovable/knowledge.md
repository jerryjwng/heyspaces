# HeySpaces Design System

## Colors

Page base: #FAFAF8
Hero/alternate bg: #F0EDE8
Surface/cards: #FFFFFF
Dark sections: #0C0C0C
Neutral: #F5F5F3
Text primary: #0C0C0C
Text secondary: #6B6B6B
Text muted: #A8A8A8
Border subtle: #EBEBEB
Border default: #E0DDD8

## Typography

Font: Inter
Hero: 72px, weight 700, tracking -2px
H1: 56px, weight 700, tracking -1.5px
H2: 44px, weight 700, tracking -1px
H3: 28px, weight 600
Body: 16px, weight 400, line-height 1.6
Small: 14px, color #6B6B6B
Label: 11px, weight 600, uppercase, tracking 0.1em, color #A8A8A8

## Components

Buttons: always border-radius 100px
Primary: bg #0C0C0C, color #FFFFFF
Ghost: border 1.5px solid #0C0C0C
Cards: bg #FFFFFF, border 1px solid #EBEBEB, border-radius 16-20px
Inputs: bg #F5F5F3, border-radius 10px
        focus: bg #FFFFFF, border #0C0C0C
Badges: border-radius 100px, font-size 11px, weight 600

## Spacing

Section padding: 120px 48px desktop / 80px 24px mobile
Max-width: 1200px centered
Card padding: 28-32px
Gap between cards: 24px

## Status Colors

Green: bg #E8F5EE, text #1A6B3C
Blue: bg #E8F0FE, text #1A3FAB
Yellow: bg #FEF6E4, text #8A5A00
Red: bg #FEE8E8, text #9B1C1C

## Rules

- Always use Inter font
- Always pill-shaped buttons (rounded-full / 100px)
- No gradients
- No colored shadows
- Always German language
- Reuse existing components
- Never invent new patterns
- Use semantic Tailwind tokens from index.css (already mapped to these values)
